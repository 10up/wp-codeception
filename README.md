# WP Codeception

## Installation

```
sudo apt-get install openjdk-6-jre-headless

npm install
composer install
```

## Commands

### Selenium

```
wp selenium start
```

```
wp selenium stop
```

### Codeception

```
wp codeception run
```

```
wp codeception bootstrap
```

```
wp codeception generate-cest
```

```
wp codeception generate-cept
```
